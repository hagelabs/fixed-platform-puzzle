## Short description (≤150 chars)

## Long description (≤1000 chars)
Test your logic skills in this strategic puzzle game. Arrange and remove blocks from the grid in the correct order. One wrong move can trap you — plan carefully!

Features:
- 50 progressively harder levels
- No time pressure
- Satisfying removal mechanics with ASMR sounds
- Mobile, tablet, and desktop
- Auto-save progress

Perfect for casual puzzle fans.

## Capture workflow
1. `npm run build && npm run dev`
2. Open http://localhost:3000 in Chrome
3. Resize devtools viewport to 1280x720
4. DevTools > Capture screenshot (Cmd+Shift+P → "Capture screenshot")
5. Save into `marketing/screenshots/`
